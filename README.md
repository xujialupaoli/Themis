# Themis

**Themis** is a robust and accurate species-level metagenomic profiler.


Themis is released under the **GPL-3.0** license, and is compatible with ganon and ggcat licenses.

---

## Features

- **Commands**
  - `themis build-custom` to build reference databases
  - `themis profile` profiling
